# 큐레이션 기준 — 후보 수집·선별

## 편집 기준점 (레퍼런스 채널)

`artart.today` · `cultureart4u`(널 위한 문화예술) · `artinculture`(아트인컬처) · `lingrongdang`(링롱댕) ·
`b.framemag` · `design.co.kr` · `eyesmag.com` · `careet.net`

이 계정들이 "오늘 올릴 법한" 소식만 고른다. 공통 감성:
- 작품/전시 뒤의 **이야기**가 되는 소식 (왜 재밌는지 한 문장으로 풀리는 것)
- MZ가 저장·공유하고 싶은 **비주얼**
- 니치하지만 힙한 취향, 셀럽·팝컬처·브랜드와 아트의 접점

검색으로 최소 2개 채널의 최근 동향 확인 (화제성 검증):
```
artart.today 최근
널 위한 문화예술 cultureart4u 최근
아트인컬처 전시
site:design.co.kr 전시
```

## 검색 방법

오늘 날짜(M월, 가능하면 D일)를 쿼리에 반드시 포함. 카테고리별 2~3건, 총 10건 이상 수집.

```
전시 개막 [M월] [연도]
아트페어 갤러리 오픈 [M월]
브랜드 아트 콜라보 팝업 [M월]
한국 작가 해외 전시 수상 [M월]
아이돌 배우 전시 미술 [연도]
```
