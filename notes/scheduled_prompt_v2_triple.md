## STEP 0 — 발행 이력 체크 및 금지 키워드 추출

```bash
python3 -c "
import json, glob, os
from datetime import date, timedelta

cutoff = date.today() - timedelta(days=30)
files = sorted(glob.glob('output/news/*.json'), key=os.path.getmtime, reverse=True)
print('=== 최근 30일 발행 이력 ===')
banned = []
for f in files:
    d = json.load(open(f, encoding='utf-8'))
    pub = d.get('published_at', '2000-01-01')
    if pub >= cutoff.isoformat():
        h1 = d.get('headline1', '')
        title = d.get('news_title', '')
        banned.append(h1)
        print(f'  {pub} | {h1} | {title[:50]}')
print()
print('=== ❌ 금지 키워드 목록 (아래 주제는 오늘 절대 선정 불가) ===')
for b in banned:
    print(f'  ❌ {b}')
"
```

- 금지 키워드와 겹치는 후보는 검색 결과에 나와도 즉시 제외. 예외 없음.
- 같은 작가·전시명·기관·페어는 30일 내 1회만.

## STEP 2 — 레퍼런스 채널 동향 파악 (필수, 나침반으로 사용)

아래 채널을 검색해 "지금 아트씬에서 무엇이 화제인가" 파악. 최소 3개 이상 검색.

```
artart.today 2026 전시 화제          ← MZ 트렌드 (76만 팔로워)
b.framemag 2026 아트 전시            ← 컬렉터·인사이더 큐레이션 톤
site:eyesmag.com 2026 전시 아트      ← 라이프스타일×아트
site:design.co.kr 2026 전시          ← 디자인·아트 웹매거진

```

**파악 목적:**
- 이 채널들이 이미 다룬 주제 → 화제성 검증 완료, 우선 후보
- 이 채널들이 아직 안 다룬 주제 → 선점 가능, 가점
- 이 채널들의 최근 톤·포맷 → 오늘 캡션 톤 참고

## STEP 4 — 후보 심층 검증 (각 후보별 3분 투자)

수집한 후보 중 살아남은 것들을 대상으로 아래를 확인:

### 4-1. 사실 교차 검증
- 기사 최소 2개 이상에서 같은 내용 확인
- 개막일·장소·작가명·주최기관 정확한지 검증
- 가능하면 공식 웹사이트 또는 미술관 공식 보도자료 1개 포함

### 4-2. 이미지 퀄리티 사전 체크
각 후보 기사 URL이 화이트리스트 도메인인지 확인:
- ✅ 화이트리스트: `biz.heraldcorp.com`, `heraldcorp.com`, `news1.kr`, `mk.co.kr`, `joongang.co.kr`, `chosun.com`, `kmib.co.kr`, `design.co.kr`
- ❌ 블랙리스트: `*.go.kr`, `kh.or.kr`, `korea.kr`, `sema.seoul.go.kr` 및 모든 공공기관 도메인

화이트리스트 도메인 기사가 없으면 → 같은 소식을 다룬 화이트리스트 기사를 별도 검색.

### 4-3. 화제성 검증
- SNS에서 실제 반응이 있는지 검색 (`인스타그램 [전시명]`, `[작가명] 화제` 등)
- artart.today·b.framemag가 이미 다뤘다면 화제성 검증 완료로 간주

### 4-4. 후보 퀄리티 점수 계산 (아래 기준, 내부 판단용)

| 항목 | 만점 | 감점 조건 |
|------|------|-----------|
| **기사 신선도** | 15점 | 오늘·어제: 15점 / 3~7일: 10점 / 8~14일: 5점 / 15일 이상: 탈락 |
| **미사용 여부** (최근 14일 JSON에 없음) | 25점 | 동일 이벤트 중복: 즉시 탈락 |
| MZ 화제성 (SNS 반응·셀럽 연관·유행) | 25점 | |
| 시각적 임팩트 (썸네일 강도) | 20점 | |
| 레퍼런스 채널 친화도 | 10점 | |
| 카테고리 다양성 기여 | 5점 | |

70점 이상인 후보만 최종 선정 대상. 14일 이내 기사면 신선도 감점을 감수하고 선정 가능. 단, 동일 이벤트 중복은 예외 없이 탈락.

### 5-1. 최종 선정 체크리스트
- [ ] 3건 모두 Step 0 금지 키워드 미충돌
- [ ] 3건이 서로 다른 카테고리
- [ ] 각 후보 퀄리티 점수 70점 이상
- [ ] image_url이 화이트리스트 도메인
- [ ] 3건의 주제·톤이 다양 (전시·콜라보·글로벌 조합 등)

### 5-2. 헤드라인 작성 (각 건별 2~3개 버전 작성 후 최선 선택)

규칙:
- 2줄, 각 줄 **6~11자**
- 1줄: 기관·전시명·아티스트명 (명사형)
- 2줄: 사실형 동사 (`개막`, `개관`, `공개`, `개최`, `선정`, `수상`)
- 자극형·감탄형 금지 ("역대급", "놀라운" 등)

**버전 예시:**
```
버전 A: 김윤신 회고전 / 호암미술관 개최
버전 B: 호암미술관 / 김윤신 60년 화업
→ 버전 A 선택 (정보 밀도 더 높음)
```

## STEP 7 — 캡션 작성 및 자기 검토 (3건 각각)

### 작성 기준
- 해요체, 큐레이터 친구 라인
- 2~3문장. 첫 문장에 "왜 지금 봐야 하는가"가 담길 것
- 정보 나열 ❌ → 시선·관점 1줄 필수
- 장소·일정은 마지막에 간결하게
- 해시태그 5개, 마지막은 항상 `#아트매거진`

### 작성 후 자기 검토 (필수)
캡션을 다 쓴 뒤 스스로에게 질문:
1. "이걸 읽고 친구한테 보내고 싶나?" → No이면 다시 쓸 것
2. "보도자료처럼 들리는 문장이 있나?" → 있으면 구어체로 교체
3. "합니다·됩니다 체가 섞였나?" → 섞였으면 해요체로 통일

**좋은 예:**
> 91세에 이런 에너지가 나올 수 있어요. 아르헨티나에서 60년을 살며 나무와 돌로 쌓아온 김윤신의 세계—재료와 작가의 경계가 사라진 100여 점이 호암미술관에 걸렸어요. 생각보다 훨씬 강렬한 전시예요.

**나쁜 예:**
> 국내외에서 주목받는 조각가 김윤신의 회고전이 호암미술관에서 개최됩니다. 60년에 걸친 방대한 작품을 만나보실 수 있습니다. (보도자료 톤, 합니다체)
